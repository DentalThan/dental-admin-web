Brief Overview

Oreo Laravel Admin Template is a popular open source WebApp Laravel template for Laravel admin dashboards and admin panels. It’s responsive Admin template, which is based on Laravel + Bootstrap 4X framework. It utilizes all of the Bootstrap components integrated with Laravel layout standards and re-styles many commonly used plugins to create a consistent design that can be used as a Larvel backend applications.

Installation

1. Download

Download the theme files from your account and place on your server.

2. Composer

Laravel project dependencies are managed through the PHP Composer tool. The first step is to install the depencencies by navigating into your project in terminal and type following command: composer install

3. NPM/Yarn

For NPM, you have to run following command from the root of the project in terminal: npm install

4. Change Environment Configuration Files

This package ships with a .env. file in the root of the project. Change/Update your desired variables in this file.

Reference: https://laravel.com/docs/5.7/installation

Folder Structure

We follow standard Laravel fodler structure and other created files are as following

Template assets files can be located at
/public/assets/*
Routes can be located at
/routes/web.php
Controller files can be located at
/app/Http/Controllers/*
View files can be located at
/resources/views/*
Layout files can be located at
/resources/views/layout/master.blade.php
/resources/views/layout/authentication.blade.php
Navigation Bar and Sidebar can be located at
/resources/views/layout/navbar.blade.php
/resources/views/layout/sidebar.blade.php
Default Error page can be locate at
/resources/views/errors/404.blade.php
